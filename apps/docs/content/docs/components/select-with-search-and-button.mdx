## Examples

### Inline creation

Wire `onButtonClick` to dynamically add a new option — the list expands immediately without leaving the dropdown.

<ComponentPreview name="select-with-search-and-button/with-action" />

### Error state

Combine `error` and `required` with a helper message to communicate validation failures clearly.

<ComponentPreview name="select-with-search-and-button/error" />

### Disabled

The trigger is non-interactive and dimmed when `disabled` is set; the action button also becomes unavailable.

## Examples

### With icons

Use a home icon as the root link and file-type icons on each crumb to communicate the type of node at a glance.

<ComponentPreview name="breadcrumb/with-icons" />

### With dropdown

Drop a `DropdownMenu` into a `BreadcrumbItem` to expose sibling pages from the trail without leaving the current view.

<ComponentPreview name="breadcrumb/with-dropdown" />

### With ellipsis

Use `BreadcrumbEllipsis` to collapse a long navigation trail and keep the bar compact.

title: Input With Inner Tags

## Examples

### Pre-filled tags

Start with an existing set of tags; users can add more by typing and pressing Enter, or remove any with the × button.

<ComponentPreview name="input-with-inner-tags/skills" />

### Empty state

An empty field showing the placeholder — type a value and press Enter to create the first tag.

<ComponentPreview name="input-with-inner-tags/empty" />

### Disabled

Existing tags are visible but no new ones can be added or removed when the field is disabled.

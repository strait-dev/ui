## Examples

### Stacked panels

Set `stacked` to join `FramePanel` children edge-to-edge with shared borders so the group reads as one continuous surface.

<ComponentPreview name="frame/stacked" />

### With footer

Use `FrameFooter` inside a panel to anchor action buttons below the content with a dividing border.

<ComponentPreview name="frame/with-footer" />

### Ghost variant

`variant="ghost"` removes the outer border; each `FramePanel` then carries its own border independently.

<ComponentPreview name="frame/ghost" />

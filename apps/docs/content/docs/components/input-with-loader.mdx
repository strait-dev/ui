## Examples

### With start icon

A static leading icon in the start slot — useful for search bars and filtered inputs at rest.

<ComponentPreview name="input-with-loader/with-start-icon" />

### Async loading state

The spinner replaces the leading icon while typing, simulating an async search or validation operation.

<ComponentPreview name="input-with-loader/loading-state" />

### With clear button

A clickable trailing icon lets users clear the input; the end icon only appears when there is text to clear.

## Examples

### Vertical orientation

Set `orientation="vertical"` on `Stepper` to stack steps top-to-bottom; pair with Back and Next buttons to drive a wizard flow.

<ComponentPreview name="stepper/vertical" />

### Sizes

The `size` prop scales both the circular indicator (`size-8` for `default`, `size-6` for `sm`) and the label text so the stepper fits in tight or spacious layouts.

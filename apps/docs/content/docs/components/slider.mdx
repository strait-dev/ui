## Examples

### Sizes

The `size` prop controls both track thickness and thumb diameter: `sm` (thin), `default`, and `lg` (thick).

<ComponentPreview name="slider/sizes" />

### Variants

The `variant` prop colours the filled range indicator and the thumb border/ring using semantic tokens.

<ComponentPreview name="slider/variants" />

### Range slider

Pass a two-element array to `defaultValue` to render two thumbs and let users select a min–max range.

<ComponentPreview name="slider/range" />

### Disabled

Set `disabled` to prevent interaction; the thumb opacity is reduced automatically.

<ComponentPreview name="slider/disabled" />

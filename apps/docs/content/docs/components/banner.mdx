## Examples

### Variants

All four intent variants — `info`, `success`, `warning`, and `destructive` — each auto-select a matching icon from context.

<ComponentPreview name="banner/variants" />

### With actions

Pair `BannerActions` with a `Button` and a `BannerClose` to give users a call-to-action alongside a dismiss control.

<ComponentPreview name="banner/with-actions" />

### Full-width layout

Use `layout="full-width"` for edge-to-edge strips at the top of a page or layout container — no side borders, no border-radius.

<ComponentPreview name="banner/full-width" />

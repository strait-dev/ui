## Examples

### Trigger variants

`variant` controls the trigger style: `default`, `muted` (dimmer), and `ghost` (underline on hover).

<ComponentPreview name="relative-time-card/variants" />

### Multiple timezones

Pass an array of IANA timezone identifiers to `timezones`; the viewer's local zone is always appended as the last row.

<ComponentPreview name="relative-time-card/multi-timezone" />

### In a feed

Embed the trigger inline in a list of events to surface relative timestamps without taking up layout space.

<ComponentPreview name="relative-time-card/feed" />

## Examples

### Variants

Three visual styles are available: `default` (flush dividers), `outline` (bordered card per item), and `solid` (filled muted card per item).

<ComponentPreview name="accordion/variants" />

### Multiple open items

Set `type="multiple"` to allow several sections to be open simultaneously; pass an array to `defaultValue` to pre-open them.

<ComponentPreview name="accordion/multiple" />

### Indicators

Switch the collapse/expand glyph between `chevron` (default) and `plus-minus` via the `indicator` prop on `AccordionTrigger`.

<ComponentPreview name="accordion/indicators" />

## Accessibility

`Switch` is built on Base UI's Switch primitive, which renders a button with `role="switch"` and `aria-checked` toggling between `true` and `false` automatically — the on/off state is announced to screen readers without any extra markup.

### Keyboard interactions

| Key | Action |
|-----|--------|
| `Space` | Toggles the switch between on and off |
| `Tab` | Moves focus to the switch (or away from it) |

### ARIA and labeling

Pair the `Switch` with a visible `<Label>` whose `htmlFor` matches the switch's `id`. When used in a settings-list layout where the label is a separate block element, use `aria-labelledby` pointing at the heading text element. For icon-only or ambiguous contexts, provide an `aria-label` directly on the `Switch`.

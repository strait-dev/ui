## Examples

### States

The switch renders off, on, disabled off, and disabled on states driven by Base UI's data attributes — no extra classes needed.

<ComponentPreview name="switch/states" />

### Sizes

Set `size` on the root to scale both the track and the thumb together: `sm` (24 × 14 px), `default` (32 × 18 px), `lg` (44 × 24 px).

<ComponentPreview name="switch/sizes" />

A common composition: place each switch at the end of a row with a label and short description on the left.

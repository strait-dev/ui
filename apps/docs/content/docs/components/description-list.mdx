## Examples

### Vertical layout

The default `orientation="vertical"` stacks each term/details pair; pairs are separated by a vertical gap.

<ComponentPreview name="description-list/vertical" />

### Horizontal with dividers

`orientation="horizontal"` places terms in a left column and details in the right; `divided` adds a subtle border between rows.

<ComponentPreview name="description-list/horizontal" />

### Size scale

Three size presets — `sm`, `default`, and `lg` — control text size and row gap progressively.

<ComponentPreview name="description-list/sizes" />

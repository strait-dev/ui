## Examples

### Variants

Three visual styles — `default`, `secondary`, and `outline` — control the tag surface and border.

<ComponentPreview name="tag-group/variants" />

### Selectable

Set `selectionMode="multiple"` (or `"single"`) to make tags toggle-selectable with keyboard and pointer.

<ComponentPreview name="tag-group/selectable" />

### Removable

Pass `onRemove` to the group to render a dismiss button on each tag; clicking it (or pressing Backspace/Delete) fires the callback with the removed keys.

<ComponentPreview name="tag-group/removable" />

description: Floating toolbar that surfaces bulk actions when one or more table or list rows are selected.

## Examples

### Icon actions

Actions that include an `icon` render as compact icon-only buttons with a tooltip revealing the label on hover.

<ComponentPreview name="bulk-action-bar/icon-actions" />

### Label actions

Actions without an `icon` render as labelled buttons — useful when the action name needs to be immediately visible.

<ComponentPreview name="bulk-action-bar/label-actions" />

### Table selection context

Place the bar inline below a table with `position="static"` and show it only when rows are selected.

<ComponentPreview name="bulk-action-bar/table-context" />

## Examples

### Nested object

Deeply nested API responses can be explored by depth with `defaultExpanded`; the `copyable` prop adds a clipboard button in the top-right corner.

<ComponentPreview name="json-viewer/nested" />

### Array of objects

Pass an array as `data` to inspect list responses — common for paginated API results.

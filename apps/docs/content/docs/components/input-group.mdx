---
title: Input Group
description: A composite control that visually merges an input with leading and trailing addons, icons, buttons, or labels into a single bordered group.
---

<ComponentPreview name="input-group/demo" />

## Examples

### Text addons

Use `InputGroupText` inside `InputGroupAddon` for currency symbols, protocol prefixes, units, or icon decorations.

<ComponentPreview name="input-group/text-addons" />

### Button addons

Place `InputGroupButton` inside an `InputGroupAddon` for compact action buttons such as search submit or copy-to-clipboard.

<ComponentPreview name="input-group/button-addons" />

### States

Disabled and invalid states lift visually from the inner control to the group's outer border via CSS `has-[]` selectors.

<ComponentPreview name="input-group/states" />

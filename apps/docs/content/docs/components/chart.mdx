## Examples

### Bar chart

A `BarChart` groups series into clustered bars by default; each series is colored from the `chart-1…chart-5` token palette.

<ComponentPreview name="chart/bar" />

### Line chart

`LineChart` connects data points with smooth lines, useful for showing trends over time.

<ComponentPreview name="chart/line" />

### Donut chart

`DonutChart` renders a hollow-centre pie with a centred total label, suited to part-to-whole breakdowns.

<ComponentPreview name="chart/donut" />

## Examples

### Vertical list

The default `strategy="vertical"` stacks items top-to-bottom; use a `SortableItemHandle` to restrict dragging to the grip icon.

<ComponentPreview name="sortable/vertical" />

### Grid

Set `strategy="grid"` and wrap the `Sortable` in a CSS grid container to drag items across a two-dimensional layout.

<ComponentPreview name="sortable/grid" />

### Add and remove items

Combine `Sortable` with add/remove controls for a fully dynamic ordered list.

<ComponentPreview name="sortable/with-remove" />

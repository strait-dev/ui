## Examples

### Variants

`TabsList` supports three visual styles: `default` (filled pill bar), `line` (borderless with an underline indicator), and `underline` (plain text with a full-width bottom border).

<ComponentPreview name="tabs/variants" />

### With icons

Place a `HugeiconsIcon` before the label inside `TabsTrigger` — the trigger's gap and icon sizing are handled automatically.

<ComponentPreview name="tabs/with-icons" />

### Vertical orientation

Set `orientation="vertical"` on `Tabs` to rotate the list into a column with full-width, left-aligned triggers.

<ComponentPreview name="tabs/vertical" />

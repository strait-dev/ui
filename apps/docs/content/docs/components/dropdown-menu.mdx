## Examples

### Checkbox items

Use `DropdownMenuCheckboxItem` with `checked` and `onCheckedChange` to build a toggleable options menu.

<ComponentPreview name="dropdown-menu/with-checkbox-items" />

### Radio group

Wrap `DropdownMenuRadioItem`s in a `DropdownMenuRadioGroup` to create a mutually exclusive selection.

<ComponentPreview name="dropdown-menu/with-radio-group" />

### Submenu

Nest a `DropdownMenuSub` with `DropdownMenuSubTrigger` and `DropdownMenuSubContent` to add a fly-out submenu.

<ComponentPreview name="dropdown-menu/with-submenu" />

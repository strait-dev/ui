## Examples

### All statuses

Every built-in status keyword maps to a semantic colour and a humanised label — unknown keywords degrade gracefully to a neutral grey pill.

<ComponentPreview name="status-badge/all-statuses" />

### Sizes

The `size` prop cascades to the leading dot so badge and dot stay proportional at every scale.

<ComponentPreview name="status-badge/sizes" />

### With pulse

Add `pulse` for an `animate-ping` ring behind the dot — use it for genuinely real-time states like `running` or `active`.

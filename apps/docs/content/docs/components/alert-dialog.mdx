## Examples

### Destructive confirmation

Use `accent="destructive"` on `AlertDialogHeader` to tint the title text with the destructive token — ideal for irreversible deletion flows.

<ComponentPreview name="alert-dialog/destructive" />

### With media

Add `AlertDialogMedia` inside the header to provide an icon or illustration that gives visual context to the action.

<ComponentPreview name="alert-dialog/with-media" />

### Sizes

The `size` prop on `AlertDialogContent` controls the max-width of the panel — `sm` renders a two-column footer grid.

<ComponentPreview name="alert-dialog/sizes" />

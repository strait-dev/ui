## Accessibility

The root renders a `<nav>` with `aria-label="pagination"` so screen readers announce it as a distinct navigation landmark. `PaginationLink` sets `aria-current="page"` on the active page link. `PaginationEllipsis` carries an `sr-only` "More pages" label and is otherwise hidden from the accessibility tree.

| Key | Action |
| --- | --- |
| `Tab` | Move focus through page links, Previous, and Next controls |
| `Shift+Tab` | Move focus backward |
| `Enter` | Navigate to the focused page link |

**ARIA:** `PaginationPrevious` and `PaginationNext` both carry explicit `aria-label` attributes (`"Go to previous page"` / `"Go to next page"`), so icon-only display on small screens still has a meaningful accessible name. Ensure each `PaginationLink` has a unique, visible number as its label so the active page is unambiguous.

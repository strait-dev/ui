## Accessibility

`Calendar` and `RangeCalendar` are built on React Aria Components, which fully manage ARIA roles, focus, and keyboard navigation in accordance with the ARIA grid/calendar pattern.

| Key | Action |
|---|---|
| `Arrow Left` / `Arrow Right` | Move focus one day backward / forward |
| `Arrow Up` / `Arrow Down` | Move focus one week backward / forward |
| `Page Up` / `Page Down` | Navigate to the previous / next month |
| `Home` | Move focus to the first day of the current month |
| `End` | Move focus to the last day of the current month |
| `Enter` / `Space` | Select the focused day (or set the range start/end) |

The root element receives `role="application"` with an `aria-label` derived from the visible heading. Use the `aria-label` or `aria-labelledby` prop on the `Calendar` / `RangeCalendar` component to provide a meaningful name when the heading alone is insufficient.

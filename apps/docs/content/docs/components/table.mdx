## Examples

### Striped rows

The `striped` variant applies an alternating `bg-muted/50` background to even body rows.

<ComponentPreview name="table/striped" />

### Bordered cells

The `bordered` variant adds a border to every `th` and `td`, creating a grid-style layout. Pair it with `TableFooter` for a summary row.

<ComponentPreview name="table/bordered" />

### Sizes

The `size` prop (`sm`, `default`, `lg`) controls cell height and padding for the whole table.

<ComponentPreview name="table/sizes" />

## Examples

### Variants

The full colour spectrum — solid fills, tinted `*-light` fills, and bordered `*-outline` options — is available through the `variant` prop.

<ComponentPreview name="badge/variants" />

### Sizes

Five height presets (`xs` through `xl`) control padding and font size via the `size` prop.

<ComponentPreview name="badge/sizes" />

### With icon

Use `iconLeft` or `iconRight` to attach a `HugeiconsIcon` glyph inside the badge without extra markup.

<ComponentPreview name="badge/with-icon" />

### Dot indicator

Add a leading presence dot with `dot`; it inherits the variant's text colour via `bg-current`.

<ComponentPreview name="badge/dot" />

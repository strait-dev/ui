## Examples

### Segmented control

Use text labels to build a period or view switcher where only one option is active at a time.

<ComponentPreview name="button-group/segmented" />

### Split button and text addon

Pair a primary action with an overflow trigger via `ButtonGroupSeparator`, or attach a static label with `ButtonGroupText`.

<ComponentPreview name="button-group/split-button" />

### Vertical orientation

Set `orientation="vertical"` to stack controls in a column with the same shared-border treatment.

<ComponentPreview name="button-group/vertical" />

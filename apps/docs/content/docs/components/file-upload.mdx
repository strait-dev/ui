## Examples

### Multiple files

Set `multiple` to allow selecting and managing several files at once.

<ComponentPreview name="file-upload/multiple" />

### Sizes

Three dropzone sizes — `sm`, `default`, and `lg` — adjust the visual weight of the drop area.

<ComponentPreview name="file-upload/sizes" />

### Validation

Use `accept` to restrict file types and `maxSize` (in bytes) to reject oversized files; the first rejection is shown below the dropzone.

<ComponentPreview name="file-upload/validation" />

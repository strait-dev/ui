## Examples

### Variants

Choose between `default` (1 px ring), `outline` (visible border), and `ghost` (transparent) surface styles.

<ComponentPreview name="card/variants" />

### With footer and action

Compose `CardFooter` and `CardAction` to add a pinned toolbar and a top-right control within the header grid.

<ComponentPreview name="card/with-footer" />

### Sizes

The `size` prop (`sm`, `default`, `lg`) cascades gap and padding to every sub-part from a single root prop.

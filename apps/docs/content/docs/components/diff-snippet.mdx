## Examples

### Variants

Use `default` for cards, `muted` for dense docs, `terminal` for dark code surfaces, and `minimal` when the surrounding container already provides chrome.

<ComponentPreview name="diff-snippet/variants" />

### Row states

Rows support `add`, `remove`, `context`, `info`, `warning`, and `error` states.

<ComponentPreview name="diff-snippet/state-matrix" />

Set `wrap` and `maxHeight` for long generated diffs in narrow containers.

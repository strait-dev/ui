---
title: Components
description: Browse the Strait UI component library.
---

Strait UI ships 120+ accessible, tree-shakeable components. Each is published as
its own subpath export (`@strait/ui/components/<name>`) and documented here with
live previews and an auto-generated props table.

Pick a component from the sidebar to get started, or see
[Installation](/docs/installation) to add the library to your project.

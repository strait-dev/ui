## Examples

### Sizes

Set `size` once on `RadioGroup` and it cascades to the circle diameter and inter-item spacing for all child items.

<ComponentPreview name="radio-group/sizes" />

### States

The group handles default selection, a fully disabled group, and individual disabled items without additional styling.

<ComponentPreview name="radio-group/states" />

### With descriptions

Pair each `RadioGroupItem` with a label and a short description paragraph to give users more context per option.

<ComponentPreview name="radio-group/with-descriptions" />

### Controlled

Pass `value` and `onValueChange` to the root to drive the selection from external state and react to changes.

<ComponentPreview name="radio-group/controlled" />

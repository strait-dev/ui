## Examples

### Range selection

Pass `mode="range"` and a `DateRange` state to let the user pick a start and end date.

<ComponentPreview name="calendar/range" />

### Multiple selection

Pass `mode="multiple"` to allow picking any number of non-contiguous dates.

<ComponentPreview name="calendar/multiple" />

### Dropdown caption

Set `captionLayout="dropdown"` with `startMonth` and `endMonth` to replace the static header with month and year selects.

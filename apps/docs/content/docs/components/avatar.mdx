## Examples

### Sizes

Five size presets (`xs` through `xl`) cascade to every sub-part — set it once on the root `Avatar`.

<ComponentPreview name="avatar/sizes" />

### Group with overflow count

Wrap multiple `Avatar` elements in `AvatarGroup` for an overlapping stack; use `AvatarGroupCount` to display a "+N" overflow chip.

<ComponentPreview name="avatar/group" />

### Fallback

`AvatarFallback` renders initials when no image is provided or while the image is loading.

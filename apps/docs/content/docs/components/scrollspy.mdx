## Examples

### Horizontal anchor bar

Place the anchors in a horizontal `<nav>` above the scrollable content to create a tab-style section indicator; the same `data-[active=true]` styling applies.

<ComponentPreview name="scrollspy/horizontal" />

### Scroll offset

Pass an `offset` in pixels to shift the activation threshold — useful when a sticky header would otherwise obscure the section heading at the top of the viewport.

<ComponentPreview name="scrollspy/with-offset" />

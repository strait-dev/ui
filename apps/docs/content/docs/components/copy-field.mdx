## Examples

### Sensitive values

Set `sensitive` for secrets. The field masks the visible value until reveal, while copy still writes the original value.

Use `default`, `muted`, `ghost`, or `terminal` depending on the surrounding surface.

### Statuses

Use `status` to communicate verified, warning, destructive, or info context without changing the copied value.

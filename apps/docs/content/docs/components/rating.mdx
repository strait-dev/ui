## Examples

### Sizes

Three star sizes — `sm`, `default`, and `lg` — control both the star pixel size and the gap between stars.

<ComponentPreview name="rating/sizes" />

### Read-only

Pass `readOnly` to render a non-interactive display element (`role="img"`) — useful for showing review scores.

<ComponentPreview name="rating/read-only" />

### Controlled

Pass `value` and `onValueChange` to manage the selected rating from the parent component.

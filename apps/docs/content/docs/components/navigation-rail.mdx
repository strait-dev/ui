## Examples

### Badge indicator

Pass a `badge` node to `NavigationRailItem` to render a small dot at the top-right corner of the icon button — useful for unread counts or alerts.

<ComponentPreview name="navigation-rail/with-badge" />

### Right orientation

Set `orientation="right"` on `NavigationRail` to anchor the rail to the trailing edge of the layout; the border flips to the left side automatically.

<ComponentPreview name="navigation-rail/right-orientation" />

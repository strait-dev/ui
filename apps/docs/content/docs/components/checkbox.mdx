## Examples

### States

The checkbox renders unchecked, checked, disabled, and invalid states through CSS data attributes — no extra styling required.

<ComponentPreview name="checkbox/states" />

### Sizes

The `size` prop scales both the outer box and the inner check icon: `sm` (14 px), `default` (16 px), `lg` (20 px).

<ComponentPreview name="checkbox/sizes" />

### Variants

The `variant` prop tints the border and checked background: `default` uses the primary token, `destructive` uses the destructive token.

<ComponentPreview name="checkbox/variants" />

### Checkbox group

Pair checkboxes with descriptive text to let users select multiple options from a labelled list.

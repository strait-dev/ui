## Examples

### Sizes

Pass `size` once on the group and every item inherits it — from `xs` to `xl`.

<ComponentPreview name="toggle-group/sizes" />

### Multiple selection

Set `multiple` to allow any combination of items to be active simultaneously, for toolbar formatting controls.

<ComponentPreview name="toggle-group/multiple" />

### Vertical orientation

Set `orientation="vertical"` to stack items in a column with the same border-collapsing treatment.

<ComponentPreview name="toggle-group/vertical" />

## Accessibility

`ToggleGroup` is built on Base UI's ToggleGroup primitive, which renders a `role="group"` container with composite keyboard navigation — arrow keys move focus between items and each item's `aria-pressed` state is managed automatically.

### Keyboard interactions

| Key | Action |
|-----|--------|
| `Arrow Right` / `Arrow Down` | Moves focus to the next item in the group |
| `Arrow Left` / `Arrow Up` | Moves focus to the previous item in the group |
| `Space` / `Enter` | Toggles the focused item's pressed state |
| `Tab` | Moves focus into the group (lands on the active or first item), then out of the group |

### ARIA and labeling

Icon-only `ToggleGroupItem` elements must carry an `aria-label` (e.g. `aria-label="Bold"`) so screen readers identify each item. When the group represents a distinct toolbar or control set, wrap it in a `<div role="toolbar" aria-label="…">` to give the region a name. Note that `aria-orientation` is intentionally suppressed on the group because `role="group"` does not permit that attribute.

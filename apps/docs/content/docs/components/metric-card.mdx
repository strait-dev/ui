## Examples

### With sparkline

Pass a numeric `data` array to render a mini bar chart that conveys trend at a glance.

<ComponentPreview name="metric-card/with-sparkline" />

### Delta directions

The `direction` field on `delta` controls the icon and colour: `"up"` is success-accented, `"down"` is destructive-accented, and `"neutral"` uses muted grey.

<ComponentPreview name="metric-card/delta-directions" />

## Accessibility

`Select` is built on Base UI's Select primitive, which implements the ARIA listbox pattern — the trigger carries `role="combobox"` and the popup carries `role="listbox"` with `aria-expanded` managed automatically.

### Keyboard interactions

| Key | Action |
|-----|--------|
| `Enter` / `Space` | Opens the dropdown when the trigger is focused; selects the highlighted option when the list is open |
| `Arrow Down` | Opens the dropdown; moves focus to the next option |
| `Arrow Up` | Moves focus to the previous option |
| `Escape` | Closes the dropdown without changing the selection |
| Printable characters | Advances focus to the next option whose label starts with the typed character (typeahead) |
| `Tab` | Closes the dropdown and moves focus to the next focusable element |

### ARIA and labeling

Pair the `SelectTrigger` with a `<Label>` by pointing `htmlFor` at the trigger's `id`. When the trigger contains only an icon, add an `aria-label` directly on `SelectTrigger` so screen readers announce the control's purpose.

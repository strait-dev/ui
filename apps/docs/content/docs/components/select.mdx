## Examples

### Sizes

Set `size` once on the `Select` root — it cascades automatically to the trigger, popup, and every item via context.

<ComponentPreview name="select/sizes" />

### States

The trigger reflects an empty placeholder, a pre-selected value, and a disabled state without any extra styling.

<ComponentPreview name="select/states" />

### Grouped options

Use `SelectGroup`, `SelectLabel`, and `SelectSeparator` to organise a long option list into labelled sections.

<ComponentPreview name="select/grouped" />

### Controlled

Pass `value` and `onValueChange` to the root to drive the selection from external state.

<ComponentPreview name="select/controlled" />

## Examples

### With icon chip

Use `media="icon"` on `EmptyMedia` to wrap an icon in a rounded chip; add a CTA inside `EmptyContent` to guide the user toward an action.

<ComponentPreview name="empty/with-icon" />

### Icon chip variants

The `variant` prop on `EmptyMedia` tints the chip with semantic colors — pair it with context-appropriate icons.

<ComponentPreview name="empty/icon-variants" />

### Borderless

Pass `border={false}` when the empty state is already enclosed by a bordered container to avoid double borders.

<ComponentPreview name="empty/borderless" />

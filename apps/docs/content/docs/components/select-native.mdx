## Examples

### States

Default, disabled, and invalid states each apply the design-system contract: `opacity-50` for disabled and the destructive ring for `aria-invalid`.

<ComponentPreview name="select-native/states" />

### With option groups

Pass raw `<optgroup>` children to group related options under labelled sections using the native browser picker.

<ComponentPreview name="select-native/with-optgroups" />

### Multiple selection

Set `multiple={true}` to render a listbox; the chevron icon is hidden and selected options receive accent highlighting.

<ComponentPreview name="select-native/multiple" />

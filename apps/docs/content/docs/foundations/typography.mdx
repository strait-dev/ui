---
title: Typography
description: The semantic type scale, from display headings to micro text.
---

The type scale is a set of semantic steps exposed as Tailwind utilities
(`text-display`, `text-h1`, `text-body`, …). Each step carries its own size,
line-height, and weight, so headings and body copy stay consistent without
hand-tuning. The default font is Inter; set `--font-heading` to ship a distinct
heading face.

# Shadows

Elevation uses Tailwind's default shadow scale unchanged — a restrained,
low-contrast set that suits the warm-neutral surfaces. Shadows signal layering:
inputs and resting cards use the smallest steps; popovers, dropdowns, and
dialogs use the larger ones. Apply them with the `shadow-*` utilities.

<div
  style={{
    display: "grid",
    gridTemplateColumns: "repeat(auto-fill, minmax(170px, 1fr))",
    gap: 24,
    padding: "8px 4px 16px",
  }}
>
  <Box name="shadow-2xs" shadow="0 1px rgb(0 0 0 / 0.05)" />
  <Box name="shadow-xs" shadow="0 1px 2px 0 rgb(0 0 0 / 0.05)" />

  <Box

    shadow="0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1)"
  />
  <Box
    name="shadow-lg"
    shadow="0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.1)"
  />
  <Box
    name="shadow-xl"
    shadow="0 20px 25px -5px rgb(0 0 0 / 0.1), 0 8px 10px -6px rgb(0 0 0 / 0.1)"
  />
  <Box name="shadow-2xl" shadow="0 25px 50px -12px rgb(0 0 0 / 0.25)" />
</div>

Prefer the smallest shadow that still communicates the layer. Over-elevation
flattens the hierarchy — if everything floats, nothing does.

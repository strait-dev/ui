# Colors

Every colour in Strait is a semantic token defined as an `oklch` CSS custom
property in `globals.css`. Components reference the Tailwind colour utilities
that map to these tokens (`bg-primary`, `text-muted-foreground`, …) — never raw
hex or named colours. The same token names resolve to different values under
`:root` (light) and `.dark`, so theming is a single class flip.

Use the toolbar paintbrush to switch the page theme, or compare both below.

## Brand & primary

`--primary` is warm ink (used for default buttons); `--brand` is the signature
orange used for emphasis. Each has a `-foreground` for text drawn on top.

</Themes>

## Status intents

Each intent has three tokens: the solid `base` (fills), `-foreground` (text on
the solid fill), and `-accent` (readable text/border for soft and outline
emphasis on tinted surfaces). The `-accent` values are tuned to clear WCAG AA
4.5:1 on a 10%-opacity tint.

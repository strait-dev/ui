# Typography

The interface font is **Inter** (loaded as a variable font and wired to
Tailwind's `font-sans`); code and tabular figures use **JetBrains Mono**
(`font-mono`). Set type with Tailwind's size utilities — they map to the scale
below.

## Weights

Inter is variable, so any weight between 100–900 is available. The system
leans on three.

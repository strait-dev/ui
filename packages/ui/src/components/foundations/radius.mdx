# Radius

Corner rounding derives from a single base token, `--radius` (`0.625rem`, 10px),
inspired by GitBook's soft geometry. The named steps are computed as multiples
of that base, so changing one value rescales the whole system proportionally.
Use the Tailwind `rounded-*` utilities, which map to these tokens.

For pills and circles, `rounded-full` applies an effectively infinite radius —
used by avatars, badges, and toggle switches.

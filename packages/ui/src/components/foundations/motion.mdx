# Motion

Motion runs on five tokenized durations and three easing curves
(DESIGN.md §7), consumed as `duration-(--duration-*)` and the `ease-*`
utilities. Raw millisecond values are banned in component code — the
`motionToken` contract rule enforces it.

Three rules govern usage:

- Animate **transform and opacity only**.
- **Exits run one duration step faster than enters** — a popover that enters at
  `base` exits at `fast` (`duration-(--duration-base)
  data-closed:duration-(--duration-fast)`).
- Everything respects the global `prefers-reduced-motion` kill-switch.

# Icons

Icons come from [Hugeicons](https://hugeicons.com) — the free set
(`@hugeicons/core-free-icons`) rendered through the `<HugeiconsIcon>` React
component. Import the icon you need and pass it to the `icon` prop; size and
colour are inherited from the surrounding text (`size-4`, `text-muted-foreground`)
unless overridden.

```tsx
import { Search01Icon } from "@hugeicons/core-free-icons";
import { HugeiconsIcon } from "@hugeicons/react";

<HugeiconsIcon icon={Search01Icon} className="size-4" />;
```

## Sizes

Icons sit on a four-step ladder (DESIGN.md §4). Each control's `[&_svg]:size-*`
selector applies the right step automatically — never size an icon with an
arbitrary value, only with another step.

<div style={{ display: "flex", flexWrap: "wrap", gap: 16, margin: "16px 0" }}>
  {[
    ["--icon-xs", "size-3", "badges, xs buttons"],
    ["--icon-sm", "size-3.5", "sm buttons"],
    ["--icon-md", "size-4", "default+ controls, menus"],
    ["--icon-lg", "size-5", "empty states, callouts"],
  ].map(([token, tw, use]) => (
    <div
      key={token}
      style={{
        display: "flex",
        alignItems: "center",
        gap: 12,
        padding: 16,
        borderRadius: "var(--radius-lg)",
        border: "1px solid var(--border)",
      }}
    >
      <HugeiconsIcon
        icon={Settings01Icon}
        style={{ width: `var(${token})`, height: `var(${token})` }}
      />
      <div>
        <code style={{ fontSize: 12 }}>{token} · {tw}</code>
        <div style={{ fontSize: 12, opacity: 0.7 }}>{use}</div>
      </div>
    </div>
  ))}
</div>

      </span>

</div>

Icon-only controls must always carry an accessible name — pass `aria-label` to
the trigger (e.g. a `Button` with `size="icon"`), since the glyph itself is
decorative to assistive technology.
